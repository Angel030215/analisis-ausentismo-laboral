
# 📊 Análisis de Ausentismo Laboral

Dashboard interactivo desarrollado para monitorear indicadores clave de ausentismo y apoyar la toma de decisiones en entornos operativos.

---

## 🚀 Herramientas utilizadas
- Excel
- Tablas dinámicas
- Power Query

---

## 📌 Objetivo
Analizar el comportamiento del ausentismo laboral para detectar patrones, tendencias y áreas de mejora dentro de la organización.

---

## 📊 Análisis realizado
- Comparación de ausentismo por año (2023–2026)
- Análisis mensual de tendencias
- Clasificación de tipos de faltas
- Comparativa entre personal sindicalizado y no sindicalizado
- Identificación de áreas con mayor impacto

---

## 📈 Resultados
- Identificación de meses críticos con mayor ausentismo
- Detección de patrones repetitivos
- Apoyo en estrategias de reducción de ausencias
